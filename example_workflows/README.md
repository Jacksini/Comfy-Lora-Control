# Workflows de Ejemplo

Esta carpeta contiene workflows de ejemplo para demostrar el uso de los nodos personalizados de Comfy-Lora-Control.

## Archivos Incluidos

### basic_example.json
Un workflow básico que muestra cómo usar el nodo ExampleNode para procesar texto.

**Cómo usar:**
1. Abre ComfyUI
2. Arrastra el archivo `basic_example.json` a la interfaz
3. El workflow se cargará automáticamente
4. Haz clic en "Queue Prompt" para ejecutar

## Crear tus Propios Workflows

Para crear workflows de prueba para tus nodos:

1. Construye un workflow en ComfyUI usando tus nodos
2. Guarda el workflow (botón "Save" o Ctrl+S)
3. Copia el archivo JSON guardado a esta carpeta
4. Renómbralo descriptivamente

## Compartir Workflows

Los workflows son excelentes para:
- Documentar casos de uso
- Probar funcionalidad
- Compartir ejemplos con usuarios
- Debugging y desarrollo
